# Getting Started

This guide walks you through installing the Typewoo SDK and making your first API calls.

## Installation

Install the SDK and its peer dependencies using npm, yarn, or pnpm:

```bash
# npm
npm install @typewoo/sdk axios qs zod

# yarn
yarn add @typewoo/sdk axios qs zod

# pnpm
pnpm add @typewoo/sdk axios qs zod
```

## Basic Setup

### Initialize the SDK

The recommended way to initialize the SDK is using `createTypewoo()`. This returns a typed client instance:

```typescript
import { createTypewoo } from '@typewoo/sdk';

const sdk = createTypewoo({
  baseUrl: 'https://your-store.com',
});
```

You can export this instance from a shared module and import it throughout your application.

<Note>
  The legacy `Typewoo.init()` singleton pattern is deprecated and will be
  removed in a future major version. Migrate to `createTypewoo()` for better
  TypeScript support and custom endpoints.
</Note>

### Access Store Services

After initialization, access the Store API through `sdk.store`:

```typescript
// Get products
const { data: products, error } = await sdk.store.products.list();

if (error) {
  console.error('Failed to fetch products:', error.message);
} else {
  console.log('Products:', products);
}
```

## Quick Examples

### Fetch Products

```typescript
// List products with filters
const { data: products, pagination } = await sdk.store.products.list({
  per_page: 12,
  orderby: 'popularity',
  category: 'clothing',
});

// Get a single product by ID
const { data: product } = await sdk.store.products.single({ id: 123 });

// Get a single product by slug
const { data: productBySlug } = await sdk.store.products.single({
  slug: 'blue-t-shirt',
});
```

### Cart Operations

```typescript
// Get the current cart
const { data: cart } = await sdk.store.cart.get();

// Add an item to the cart
const { data: updatedCart } = await sdk.store.cart.add({
  id: 123, // Product or variation ID
  quantity: 2,
});

// Update item quantity
const { data: cart } = await sdk.store.cart.update(
  'item_key_here', // Cart item key
  3 // New quantity
);

// Remove an item
const { data: cart } = await sdk.store.cart.remove('item_key_here');

// Apply a coupon
const { data: cart } = await sdk.store.cart.applyCoupon('SAVE10');

// Remove a coupon
const { data: cart } = await sdk.store.cart.removeCoupon('SAVE10');
```

### User Authentication

Authentication requires the Typewoo WordPress plugin to be installed on your WooCommerce store.

```typescript
// Login with username and password
const { data: authResponse, error } = await sdk.auth.token({
  login: 'username',
  password: 'password',
});

if (error) {
  console.error('Login failed:', error.message);
} else {
  console.log('User:', authResponse.user.display_name);
  // Token is automatically stored and used for subsequent requests
}

// Check authentication status
const { data: status } = await sdk.auth.status();
console.log('Authenticated:', status?.authenticated);

// Logout (revoke token)
await sdk.auth.revokeToken();
```

### Checkout Flow

```typescript
// Get checkout data (cart + customer info)
const { data: checkout } = await sdk.store.checkout.get();

// Update customer addresses
const { data: updatedCheckout } = await sdk.store.checkout.update({
  billing_address: {
    first_name: 'John',
    last_name: 'Doe',
    email: 'john@example.com',
    // ... other fields
  },
  shipping_address: {
    first_name: 'John',
    last_name: 'Doe',
    // ... other fields
  },
});

// Process order and payment
const { data: orderResult, error } =
  await sdk.store.checkout.processOrderAndPayment({
    billing_address: {
      /* ... */
    },
    shipping_address: {
      /* ... */
    },
    payment_method: 'stripe',
    payment_data: [{ key: 'stripe_token', value: 'tok_xxx' }],
  });

if (orderResult) {
  console.log('Order ID:', orderResult.order_id);
  console.log('Redirect URL:', orderResult.payment_result.redirect_url);
}
```

## Listening to Events

The SDK emits events for key operations. Subscribe to them for reactive UI updates:

```typescript
// Listen for cart updates
sdk.events.on('cart:updated', (cart) => {
  console.log('Cart updated:', cart?.items_count);
});

// Listen for authentication changes
sdk.events.on('auth:changed', (isAuthenticated) => {
  console.log('Auth state:', isAuthenticated ? 'logged in' : 'logged out');
});

// Listen for loading states
sdk.events.on('cart:loading', (isLoading) => {
  if (isLoading) {
    showSpinner();
  } else {
    hideSpinner();
  }
});
```

## TypeScript Support

All API responses are strongly typed. Use TypeScript for full IntelliSense support:

```typescript
import { createTypewoo, ProductResponse, CartResponse } from '@typewoo/sdk';

const sdk = createTypewoo({ baseUrl: 'https://your-store.com' });
const { data } = await sdk.store.products.list();

// `data` is typed as ProductResponse[] | undefined
data?.forEach((product: ProductResponse) => {
  console.log(product.name, product.prices.price);
});
```
