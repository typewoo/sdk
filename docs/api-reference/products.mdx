# Products API

The Products API provides access to WooCommerce product data through the Store API.

## Access

```typescript
import { createTypewoo } from '@typewoo/sdk';

const sdk = createTypewoo({ baseUrl: 'https://your-store.com' });
const productService = sdk.store.products;
```

## Methods

### `list(params?, options?)`

Retrieve a paginated list of products.

```typescript
const { data, error, pagination } = await sdk.store.products.list({
  per_page: 12,
  page: 1,
  orderby: 'popularity',
  order: 'desc',
});
```

**Parameters:**

| Name      | Type             | Required | Description      |
| --------- | ---------------- | -------- | ---------------- |
| `params`  | `ProductRequest` | No       | Query parameters |
| `options` | `RequestOptions` | No       | Request options  |

**Returns:** `Promise<ApiPaginationResult<ProductResponse[]>>`

### `single(params, options?)`

Retrieve a single product by ID or slug.

```typescript
// By ID
const { data, error } = await sdk.store.products.single({ id: 123 });

// By slug
const { data, error } = await sdk.store.products.single({
  slug: 'blue-t-shirt',
});
```

**Parameters:**

| Name      | Type                             | Required | Description                                |
| --------- | -------------------------------- | -------- | ------------------------------------------ |
| `params`  | `{ id?: number; slug?: string }` | Yes      | Product identifier (at least one required) |
| `options` | `RequestOptions`                 | No       | Request options                            |

**Returns:** `Promise<ApiResult<ProductResponse>>`

## Query Parameters

The `ProductRequest` type supports extensive filtering:

### Pagination

| Parameter  | Type     | Default | Description           |
| ---------- | -------- | ------- | --------------------- |
| `page`     | `number` | `1`     | Page number           |
| `per_page` | `number` | `10`    | Items per page        |
| `offset`   | `number` | -       | Offset for pagination |

### Sorting

| Parameter | Type              | Default  | Description    |
| --------- | ----------------- | -------- | -------------- |
| `order`   | `'asc' \| 'desc'` | `'desc'` | Sort direction |
| `orderby` | `string`          | `'date'` | Sort field     |

Valid `orderby` values:

- `date` - Date created
- `modified` - Date modified
- `id` - Product ID
- `include` - Preserve order of IDs in `include` param
- `title` - Product title
- `slug` - Product slug
- `price` - Product price
- `popularity` - Total sales
- `rating` - Average rating
- `menu_order` - Menu order
- `comment_count` - Review count

### Filtering

| Parameter        | Type       | Description                        |
| ---------------- | ---------- | ---------------------------------- |
| `search`         | `string`   | Search term                        |
| `slug`           | `string`   | Filter by slug(s), comma-separated |
| `include`        | `number[]` | Include specific IDs               |
| `exclude`        | `number[]` | Exclude specific IDs               |
| `parent`         | `number[]` | Filter by parent IDs               |
| `parent_exclude` | `number[]` | Exclude parent IDs                 |

### Date Filters

| Parameter     | Type     | Description                       |
| ------------- | -------- | --------------------------------- |
| `after`       | `string` | Products created after (ISO8601)  |
| `before`      | `string` | Products created before (ISO8601) |
| `date_column` | `string` | Column for date filtering         |

Valid `date_column` values: `date`, `date_gmt`, `modified`, `modified_gmt`

### Product Properties

| Parameter  | Type      | Description                       |
| ---------- | --------- | --------------------------------- |
| `type`     | `string`  | Product type                      |
| `sku`      | `string`  | Filter by SKU(s), comma-separated |
| `featured` | `boolean` | Filter by featured status         |
| `on_sale`  | `boolean` | Filter by on-sale status          |
| `rating`   | `number`  | Filter by minimum rating          |

Valid `type` values: `simple`, `grouped`, `external`, `variable`, `variation`

### Category Filters

| Parameter           | Type     | Description                            |
| ------------------- | -------- | -------------------------------------- |
| `category`          | `string` | Category IDs or slugs, comma-separated |
| `category_operator` | `string` | Logical operator                       |

Valid operators: `in`, `not_in`, `and`

```typescript
// Products in clothing OR accessories
const { data } = await sdk.store.products.list({
  category: 'clothing,accessories',
  category_operator: 'in',
});

// Products in BOTH clothing AND summer
const { data } = await sdk.store.products.list({
  category: 'clothing,summer',
  category_operator: 'and',
});
```

### Tag Filters

| Parameter      | Type     | Description                       |
| -------------- | -------- | --------------------------------- |
| `tag`          | `string` | Tag IDs or slugs, comma-separated |
| `tag_operator` | `string` | Logical operator                  |

### Brand Filters

| Parameter        | Type     | Description                         |
| ---------------- | -------- | ----------------------------------- |
| `brand`          | `string` | Brand IDs or slugs, comma-separated |
| `brand_operator` | `string` | Logical operator                    |

### Price Filters

| Parameter   | Type     | Description                            |
| ----------- | -------- | -------------------------------------- |
| `min_price` | `string` | Minimum price (smallest currency unit) |
| `max_price` | `string` | Maximum price (smallest currency unit) |

**Note:** Prices use the smallest currency unit. For USD ($100.25), use `10025`.

```typescript
// Products between $10 and $50
const { data } = await sdk.store.products.list({
  min_price: '1000',
  max_price: '5000',
});
```

### Stock Filters

| Parameter      | Type       | Description            |
| -------------- | ---------- | ---------------------- |
| `stock_status` | `string[]` | Filter by stock status |

Valid values: `instock`, `outofstock`, `onbackorder`

```typescript
const { data } = await sdk.store.products.list({
  stock_status: ['instock', 'onbackorder'],
});
```

### Attribute Filters

| Parameter            | Type     | Description                 |
| -------------------- | -------- | --------------------------- |
| `attributes`         | `array`  | Attribute filter objects    |
| `attribute_relation` | `string` | Relation between attributes |

### Visibility

| Parameter            | Type     | Description               |
| -------------------- | -------- | ------------------------- |
| `catalog_visibility` | `string` | Catalog visibility filter |

Valid values: `any`, `visible`, `catalog`, `search`, `hidden`

## Examples

### Basic Product Listing

```typescript
const { data: products, pagination } = await sdk.store.products.list({
  per_page: 12,
});

console.log(`Showing ${products?.length} of ${pagination.total} products`);
console.log(`Page 1 of ${pagination.totalPages}`);
```

### Filtered Product Search

```typescript
const { data } = await sdk.store.products.list({
  search: 'shirt',
  category: 'clothing',
  min_price: '1000',
  max_price: '5000',
  stock_status: ['instock'],
  orderby: 'price',
  order: 'asc',
});
```

### Product Detail Page

```typescript
const { data: product, error } = await sdk.store.products.single({
  slug: 'blue-cotton-shirt',
});

if (error) {
  console.error('Product not found');
  return;
}

console.log('Product:', product.name);
console.log('Price:', product.prices.price);
console.log('In stock:', product.is_in_stock);
```

### Pagination

```typescript
async function loadProducts(page: number) {
  const { data, pagination } = await sdk.store.products.list({
    page,
    per_page: 12,
  });

  return {
    products: data,
    hasNext: pagination.next !== undefined,
    hasPrev: pagination.previous !== undefined,
    total: pagination.total,
  };
}
```

## Admin Products API

For full CRUD operations on products, use the Admin API:

```typescript
// List products
const { data } = await sdk.admin.products.list();

// Get single product
const { data } = await sdk.admin.products.getById(123);

// Create product
const { data } = await sdk.admin.products.create({
  name: 'New Product',
  type: 'simple',
  regular_price: '19.99',
});

// Update product
const { data } = await sdk.admin.products.update(123, {
  regular_price: '24.99',
});

// Delete product
const { data } = await sdk.admin.products.delete(123, true);

// Batch operations
const { data } = await sdk.admin.products.batch({
  create: [{ name: 'Product 1' }],
  update: [{ id: 123, regular_price: '29.99' }],
  delete: [456],
});
```
